To run the program, change the configuration to run the group.Main class and add the appropriate file paths for your SDK, then press run
Once the program is running, use a valid login stored in the MySQL database
For the additional report, I decided to use a MySQL query to show details on the various countries for appointments that are scheduled.
Using an inner join made acquiring and displaying the results simple and efficient
